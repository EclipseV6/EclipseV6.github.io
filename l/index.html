How to Use

  • Enter any valid URL in the input field above
  • Choose to either launch the URL in an about:blank tab, generate a Blob URL, or download it as a Chronos file
  • Click the Execute button to perform the selected action
  • For the about:blank option, the site will open in a new tab
  • For the Blob URL option, a Blob URL will be generated and opened in a new tab
  • For the Chronos file, an HTML file will be downloaded that embeds the site in an iframe
